Privacy Policy

Introduction

This Privacy Policy is meant to help you understand the information we collect, how we use this information and what information we share.

Information we collect

The software consists of the Trekking Maps App which makes database requests to the Trekking Maps Server on the users behalf. Database requests are required because it is not practical to bundle terabytes of data with the App. These database requests include vector map tiles, map imagery, 3D geometry, roads and labels.

The App may also request your GPS location for the purpose of showing your location on the map, however, the GPS location is NOT shared with the Server. The Server collects logs of database requests which may also include the users Internet Protocol (IP) address.

How we use this Information

The IP address collected by Server logs may be used to monitor Server health (e.g. security breaches or denial of service attacks).

The database requests collected by Server logs are aggregated to generate analytics. These analytics are used identify popular areas, improve online services and monitor Server health. These analytics do NOT include the user IP address.

What information we share

The analytics may be published via the app/website or shared with marketing/research partners.

We do not share your personal information (e.g. requests associated to an IP address) with third parties unless required by law.

We utilize LocationIQ (a third party service) for user searches.
